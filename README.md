# customer
This is a sample application using PHP and Symfony.

## Used Technologies
- Symfony 6.0.8
- PHP 8.1.5
- MySQL 8
- Nginx
- Docker
- Adminer
- Bootstrap 5

## Features
- Adding and view list of customers.

## Launch
- Clone the repo.
- Run "docker-compose up -d" from the project directory.
- Run composer install.
- Go to http://localhost:8080/.
